##  Fichiers de Données Manquants

Consultez `data/INPUTS_REQUIS.txt` dans chaque sous-projet pour la liste
complète des fichiers à récupérer des auteurs.

##  Utilisation

1. Extraire l'archive
2. Lire les fichiers `INPUTS_REQUIS.txt`
3. Placer les fichiers requis dans les dossiers `data/`
4. Installer les dépendances: `pip install pandas numpy geopandas rasterio matplotlib jupyter`
5. Lancer Jupyter: `jupyter notebook`
