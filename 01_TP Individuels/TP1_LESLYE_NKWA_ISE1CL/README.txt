# TP1 INDIVIDUEL - Ghana

* **Objectif** : Analyser et visualiser les données de population du Ghana issues de WorldPop.
* **Données** : Raster WorldPop du Ghana, comprenant la population totale par pixel.
* **Traitements réalisés** :

  * Importation des données raster
  * Visualisation des propriétés du raster
  * Calcul du nombre total de pixels, surface et périmètre
  * Visualisation cartographique par région
* **Technologies** : Python (pandas, geopandas, rasterio, matplotlib, seaborn)
* **Organisation** :

  * `data/` : données brutes et traitées
  * `scripts/` : scripts Python pour traitement et visualisation
* **Usage** : exécuter les notebooks pour reproduire les analyses et visualisations
